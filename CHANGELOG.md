# Changelog

## v0.1.0 - 2026-05-29

Initial public release.

### Added

- GNOME shortcut driven dictation daemon.
- AppIndicator tray icon with Settings window.
- OpenAI/local whisper.cpp engines.
- Wayland clipboard and optional ydotool auto-paste.
- Dictation history.
- Autostart and user systemd service support.
- Test suite and GitHub Actions workflow.

### Security

- Auto-paste and local Whisper setup are explicit installer options.
- ydotool is documented as privileged desktop input access.
